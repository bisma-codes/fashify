## 📁 Folder Structure

```bash
fashify/
├── public/
├── src/
│   ├── assets/
│   ├── components/
│   └── App.jsx
├── index.html
├── tailwind.config.js
├── vite.config.js
├── package.json
└── README.md

# Clone the repository
git clone https://github.com/bisma-codes/fashify.git
cd fashify

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
